# 项目简介

## 一句话介绍

`rag-lite-kit` 是一个面向知识库问答场景的轻量级 RAG 开源 starter，强调 **低门槛可运行、工程结构清晰、便于二次扩展**。

## 项目定位

和只展示概念的 demo 不同，这个项目更关注：
- 如何快速搭一条完整 RAG 链路
- 如何把检索、Prompt、服务接口拆分为可维护模块
- 如何让招聘方或协作者低成本把项目跑起来

## 适用场景

- 企业内部知识问答助手
- 行业垂直文档问答原型
- 大模型应用工程师求职作品
- RAG 教学或技术分享演示

## 主要价值

- 对初学者：能看懂完整闭环
- 对面试官：能看到工程化意识
- 对团队 PoC：能直接作为起始模板
